# Sicurezza

## Segnalare una vulnerabilità

Se trovi una vulnerabilità in questa landing o nel prodotto BetterFit, **non
aprire una issue pubblica con i dettagli**. Usa invece la funzione
[Report a vulnerability](https://github.com/GPOI-2526-5E/BetterFit/security)
di GitHub sul repository principale, oppure contatta i maintainer indicati
nella pagina dell'organizzazione.

Indica: componente interessato, passi per riprodurre, impatto stimato.

## Ambito

Questa repository contiene un sito statico senza backend, autenticazione o
gestione di dati personali. Le superfici rilevanti sono limitate (dipendenze
npm, contenuto servito). Il prodotto vero e proprio — backend e dashboard —
vive nel repository principale, che è il punto di riferimento anche per le
segnalazioni di sicurezza.
