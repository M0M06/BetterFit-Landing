# Codice di condotta

Questo progetto adotta il [Contributor Covenant](https://www.contributor-covenant.org/it/version/2/1/code_of_conduct/),
versione 2.1.

In sintesi: partecipa in modo rispettoso e costruttivo. Non sono tollerati
molestie, attacchi personali, linguaggio discriminatorio o comportamenti che
rendano l'ambiente ostile.

Le violazioni possono essere segnalate aprendo una issue nel
[repository principale](https://github.com/GPOI-2526-5E/BetterFit/issues) o
contattando direttamente i maintainer indicati su GitHub. Le segnalazioni
vengono gestite con riservatezza.
